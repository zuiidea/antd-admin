# 配置项

你可以在 `/src/utils/config.js` 里做一些自定义配置：

## siteName

- 类型： `String`

  配置站点名称，应用到登录框，侧边栏顶部的标题文字显示。

## copyright

- 类型： `String`

  配置版权声明，应用到登录页、`Primay`布局底部。

## logoPath

- 类型： `String`

  配置站点 Logo，应用到登录框，侧边栏顶部的 Logo 显示。

## apiPrefix

- 类型： `String`

  配置项目中接口的前缀，接口相关文档可查看 [接口配置](API-configuration.md)

## fixedHeader

- 类型： `String`

  在`Primary`布局下，页面滚动时是否固定顶部。
